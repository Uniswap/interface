### To Start Development

###### Installing dependency
```bash
yarn
```

###### Running locally on Rinkeby
```bash
yarn start:rinkeby
```

###### Running locally on other testnet
```bash
REACT_APP_NETWORK_ID=2 REACT_APP_NETWORK='Ropsten Test Network' react-scripts start
```
