# Flex

`Flex` offers horizontal or vertical spacing _between_ children.

<Canvas>
  <Story name="default" args={{}}>
    {Template.bind({})}
  </Story>
</Canvas>

`Flex` exposes the following:

- `centered` as a shorthand for `alignItems='center' justifyContent='center'`
- `gap`: spacing between children as defined in theme
- and every `BoxProps`
